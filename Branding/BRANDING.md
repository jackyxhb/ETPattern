# Branding Assets

## Logo Usage

The primary logo for English Thought (ET) is located at `Branding/logo.jpg`. This JPG image serves as the source for all app icons and marketing materials.

### App Icon Generation

The app icons are generated from this logo using `sips` command-line tool to resize to various pixel sizes required by iOS:

- 20x20@1x (20px)
- 20x20@2x (40px)
- 20x20@3x (60px)
- 29x29@1x (29px)
- 29x29@2x (58px)
- 29x29@3x (87px)
- 40x40@2x (80px)
- 40x40@3x (120px)
- 60x60@2x (120px)
- 60x60@3x (180px)
- 76x76@1x (76px)
- 76x76@2x (152px)
- 83.5x83.5@2x (167px)
- 1024x1024@1x (1024px for App Store)

### Color Palette

- Primary: ET monogram design
- Background: Gradient themes used throughout the app
- Accent: Modern button and progress bar colors

### Export Guidelines

- Always maintain aspect ratio when resizing
- Use high-quality JPG compression for web/marketing
- PNG format for app icons with transparency where needed
